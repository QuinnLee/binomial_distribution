Binomial Distribution

Welcome to an interactive binomial distribution. The PDF is a blue curve, with the left X axis. The CDF curve is black, with the right X axis. To create PDF and CDF curves, change the Dot Sample Size(N) and Grey box area (P). By running trials, dots are generated randomly and the ones in the grey box turn blue. The data from the trials are graphed on the graph below. Unfortunately, there is some rounding errors with the numbers :(
